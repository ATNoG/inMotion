Sobre o projeto

O inMotion é um sistema de investigação que usa a força do sinal WiFi (RSSI) de telemóveis para inferir padrões de movimento. Participantes ligam o dispositivo ao WiFi da sala, caminham mantendo tráfego ativo, e o sistema captura 10 amostras de RSSI para cada um.

Um modelo de machine learning (RandomForest) processa essas 10 amostras e prevê uma das classes AA, AB, BA ou BB, associadas a diferentes padrões. Esta demo foi concebida para o eXperimenta, onde alunos dos 10.º ao 12.º ano observam o processo de investigação em tempo real.
